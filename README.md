## Latex editor based on GUMMI

- Snippets to autocomplete via Keybinds.
- Real time Preview
- fast compilation
- GTK4-Adwaita framework
- sharper on HIdpi Displays 

## Meson buildable
```
$ meson setup bild
$ ninja -C build
```
